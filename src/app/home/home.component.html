The Code

You've seen the paint & wine classes, now it's time to get techy and learn web development basics over trendy cocktails.
The first thing to know is no prior programming experience is required! We can say that because we have plenty of mentors on hand to help you through the tough parts, no matter what session you sign up for.
As an intro to programming, you'll get an overview of how sites are built for the web, from basic HTML structure to handling user interaction with JavaScript. You will learn about version control and how to collaborate with other developers using GitHub. We like to think of the curriculum as the essential front-end developer tool kit.
Once you've hit the Ballmer Peak courtesy of custom crafted cocktails, it's to infinity and beyond in writing code and learning to mentor others.

The Cocktails

We couldn't be more excited to partner with bartender Laura Wagner of Logistics & Libations. She brings the freshest local ingredients together with liquors from local vendors to create a custom cocktail menu to get your code on to.
We encourage everyone to drink responsibly and allow ample time to descend from the Ballmer Peak. In addition to cocktails, we offer custom crafted non-alcoholic drinks and encourage the use of ride-share programs (such as Uber or Lyft) or taxis when needed.

The Mentors

As a woman, learning to program can be intimidating! We've recruited fantastic female mentors to create a welcoming learning environment and answer those questions you may be too scared to ask!
These ladies write code for their day job and can't wait to help you get started. They're kind of like super heroes, with great taste!
